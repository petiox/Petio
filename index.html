FAQ

Comment utiliser Petio ?
Tapez votre question dans la zone de saisie. Petio analysera les fichiers du mode actif et vous répondra instantanément. Utilisez le menu + pour changer de mode.

Que sont les "Modes" ?
Chaque mode charge une base de connaissances différente. Par exemple, le mode Novice propose des explications simples, tandis que le mode Formate contient des procédures techniques avancées. Les réponses sont strictement limitées au mode sélectionné.

Petio fonctionne-t-il hors ligne ?
Oui. Une fois l'application installée en tant que PWA et les fichiers chargés, Petio fonctionne entièrement hors ligne.

Comment obtenir un accès VIP ?
Cliquez sur "Avoir mes accès" sur l'écran de vérification ou contactez l'administrateur via WhatsApp.

Confidentialité

1. Collecte des données
Petio ne collecte aucune donnée personnelle sur des serveurs externes. Votre adresse email et votre code de transaction sont uniquement utilisés localement pour vérifier votre statut d'accès.

2. Stockage local
- L'historique de vos conversations est sauvegardé exclusivement dans le localStorage de votre appareil.
- Les fichiers de connaissances sont mis en cache localement pour un fonctionnement hors ligne.
- Vous pouvez effacer toutes vos données via le bouton "Déconnexion".

3. Sécurité
- Petio fonctionne en environnement sandbox (navigateur).
- Les liens externes s'ouvrent dans un nouvel onglet sécurisé.
- Aucune publicité, tracker ou script d'analyse n'est intégré.

4. Responsabilité
Comme indiqué dans les CGU, l'Administrateur n'est responsable en aucun cas de l'usage que vous faites des informations fournies.

À propos

Petio – Assistant Intelligent Local
Version 1.0 | Développé par Le Balzac

Petio est une application progressive (PWA) conçue pour offrir des réponses rapides, précises et privées. Contrairement aux assistants cloud, Petio ne dépend d'aucun serveur distant pour ses connaissances.

🔹 Points forts :
✅ 100% Local : Vos données et l'historique restent sur votre appareil.
🚀 Ultra-rapide : Recherche optimisée dans plus de 1000 fichiers texte.
🔒 Cloisonné : Chaque mode est isolé pour éviter les mélanges d'informations.
📱 PWA : Installable sur écran d'accueil, fonctionne hors ligne.
🎨 Personnalisable : Thème sombre/clair, nom de l'assistant modifiable.

Petio a été créé pour simplifier l'accès à l'information technique et pratique, en gardant un contrôle total sur la confidentialité et la performance. Merci de votre confiance.
